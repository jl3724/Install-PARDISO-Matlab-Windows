1. install visual studio and intel compiler
2. downloads PARDISO and make the license file pardiso.lic with the license key given by email
3. downloads pardiso-matlab zip file and unzip
4. put the .lic and .dll file in the program folder
5. open matlab, in the command window do the following (change the path according to computer set up):
# for pardisoinit
mex  -largeArrayDims...
 -LC:\Users\jl372\Documents\MATLAB\pardiso\lib -llibpardiso600-WIN-X86-64...
 -'LC:\Program Files\MATLAB\R2017b\extern\lib\win64\mingw64' -lmwlapack...
 -'LC:\Program Files\MATLAB\R2017b\extern\lib\win64\mingw64' -lmwblas...
 -'LC:\Program Files (x86)\IntelSWTools\compilers_and_libraries_2019.4.245\windows\compiler\lib\intel64' -lm...
 -output pardisoinit common.cpp matlabmatrix.cpp sparsematrix.cpp pardisoinfo.cpp pardisoinit.cpp
